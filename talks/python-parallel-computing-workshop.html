Python 3.0 Workshop: Parallel Computing

Tutorial Talk • October 8, 2022

I led a workshop session focused on practical parallel computing in Python. The discussion covered how intensive computations can be accelerated by splitting tasks into concurrent processes.

Participants explored implementation strategies and performance trade-offs, with examples aimed at helping them apply parallel methods in their own analytical and scientific workloads.
