Code Up: A Python and LaTeX workshop

Tutorial Talk • September 21, 2021

The Code Up workshop was a successful learning event where I led the Basics of Python session. The objective was to make Python fundamentals accessible and practical, especially for participants interested in scientific research and technical programming.

The session emphasized practical examples, confidence-building for beginners, and clear pathways for participants to continue their coding journey.
